# kryo-gdcrux
Kryo 5.x support for classes from [gdcrux](https://github.com/tommyettinger/gdcrux)

## Get it

Use JitPack.io as your repository (which gdx-liftoff projects already can do) and depend on:
```
implementation "com.github.tommyettinger:kryo-gdcrux:0.1.2.2"
```

This depends on Kryo 5.6.2 and libGDX 1.13.1 (or newer), as well as gdcrux 0.1.2 and crux 0.1.3 .

## Older Versions

See [kryo-more](https://github.com/tommyettinger/kryo-more) for older releases.
